# Token 经济七问：2026 年，AI 正在变成一种可以「按度计费」的商品

> 适合读者：对 AI 感兴趣的所有人
> 本文约 4000 字，阅读时间约 10 分钟

---

## 引言

2026 年 3 月，OpenAI 完成新一轮融资时披露了一个惊人数据：

> 用户每分钟调用其 API 的 Token 量超过 **150 亿**，折算下来一天是 21.6 万亿。[^1]

同一个月，中国国家数据局公布：中国日均 Token 调用量突破 **140 万亿**，两年增长 **1400 倍**。[^2]

一年前，「Token」这个词还只是 AI 圈内人的技术黑话。今天，它正在成为——借用英伟达 CEO 黄仁勋的话——**「AI 的语言和货币」**。[^3]

但 Token 真的能成为货币吗？Token 经济到底在说什么？这篇文章试图用七个问题，把这股席卷全行业的新浪潮讲清楚。

---

## 一、Token 经济到底是什么？

先拆一个容易混淆的概念。

**我们这儿说的 Token，不是区块链上的代币。**

它是 AI 处理和生成信息的基本计量单位。你向 ChatGPT 提一个问题，消耗一些 Token；它给你回答，生成一些 Token。一个 Token 大约对应一到两个汉字。

为什么这个东西值得专门开一门「经济学」？

原因很朴素：**Token 让 AI 第一次变成了一种可以计量、定价和交易的资源。**

打个比方——在电力出现之前，你能感受到电"有用"，但你没法算清楚「这盏灯亮一小时要花多少钱」。有了「千瓦时」这个计量单位，电力才变成了可以定价的商品，才有了电网、电力期货、电力市场。

Token 就是 AI 世界里的「千瓦时」。[^4]

围绕这个单位，一套全新的经济逻辑正在成形：有价格、有供需、有产业链、有国际竞争、有待解决的制度难题。这就是 Token 经济学要讨论的事。

---

## 二、全球一天到底消耗多少 Token？

先建立量级感。

- **OpenAI**：API 每分钟 150 亿 Token，日均约 **21.6 万亿**
- **谷歌 Gemini**：2025 年 9 月披露，每月处理 1300 万亿 Token，日均约 **43 万亿**
- **中国（2026 年 3 月）**：国家数据局公布，日均 **140 万亿** [^2]

140 万亿 Token 是什么概念？粗略折算，相当于每天生成约 2000 亿篇千字文章。全中国 14 亿人，人均每天「消费」上百篇 AI 生成的内容。

而这一切只是开始。摩根大通预测，中国 AI 推理 Token 消耗将从 2025 年的约 10 千万亿增长到 2030 年的 3900 千万亿，**再涨 370 倍**。[^5]

根据 IDC 的数据，2025 年中国企业级 MaaS 市场 Token 调用量达到 **1944 万亿**，2026 年预计达到约 **40,000 万亿**，同比增长约 **20 倍**。[^6]

上海交通大学安泰经管学院史占中教授指出：全球 AI 大模型周 Token 调用量已从 2025 年 3 月的 2.03 万亿飙升至 2026 年 3 月的 20.4 万亿，**一年增长超 8 倍**。[^7]

我们现在看到的，可能只是开胃菜。

---

## 三、为什么 Token 越便宜，全世界花的钱反而越多？

这是一个反直觉的事实。

2022 年，调用 GPT-3 级别模型需要 **60 美元 / 百万 Token**。到 2026 年初，同等能力的开源模型只要 **0.06 美元**——降了 **99.9%**。[^4]

但同一时期，全球企业 AI 云支出从 115 亿美元涨到了 370 亿美元，翻了三倍多。[^4]

为什么降价反而没省钱？

经济学家对此不感到意外——这是经典的 **杰文斯悖论**（Jevons Paradox）：当一种资源的使用效率大幅提升，总消耗量不降反升。蒸汽机效率提升后，煤炭消耗反而飙升，同一个道理。[^4]

Token 价格下降释放了被成本约束抑制的潜在需求：
- 60 美元 / 百万 Token 时——只有金融分析、药物发现这些最高价值的任务用得起
- 0.06 美元 / 百万 Token 时——代码审查、实时客服、个性化教育、AI 角色扮演都变得经济可行

每个新场景，都是新的 Token 消耗。

而故事的更深一层在于：**单个 Token 的生产成本在下降，但单个「答案」的成本可能在上升**——因为更好的答案需要更多 Token、更大的模型、更深的推理。用户买的不是 Token，是答案。

---

## 四、真正炸裂的变量：从「人用 AI」到「AI 自己用 AI」

过去两年，Token 需求增长主要靠两件事：C 端用户把 ChatGPT 当日常工具，企业把大模型嵌入业务流程。但这两个驱动力有一个共同的天花板——**人脑**。无论 AI 多便宜，人一天能读的字数是有限的。

**2025 年底开始，Agent 改变了这个格局。**

Agent 不是聊天机器人，而是能自主执行任务的 AI 程序。你给它一个目标（「帮我订最便宜的机票」），它自己去搜索、比较、填表、付款。整个过程不断调用模型、消耗 Token，**完全不需要人类注意力参与**。

一个企业部署 1000 个 Agent，每个每天消耗 100 万 Token，一年就是 3650 亿 Token。相当于一个中等国家的全体人类用户总消耗量。[^4]

史占中教授把 Token 需求增长分为三个阶段：[^7]

1. **人类用户驱动阶段（2023—2027）**：增长稳定，但受限于人类注意力的天然上限
2. **人 + Agent 共存阶段**：Agent 开始 7×24 小时运行，需求曲线从 S 型变为 J 型
3. **Agent 主导阶段**：机器自消费远超人类消费

我们当前正处于从阶段 1 到阶段 2 的拐点。2026—2028 年很可能出现一次真正意义上的「J 型爆发」。

更值得关注的是，Agent 不光消耗 Token。**已经有实验项目让 Agent 拥有自己的钱包**，自主接任务、赚收入、再用收入购买更多 Token。AI 正在从「工具」变身「经济主体」。[^4]

这也带出了全新的制度难题：Agent 的收入算谁的？它签的合同有没有法律效力？造成的损失谁来承担？

## 五、中国和美国，谁的 Token 更「值钱」？

2026 年初，一件事让硅谷措手不及。

全球最大 AI 模型 API 聚合平台 OpenRouter 数据显示：**中国模型周 Token 调用量连续三周超越美国**，全球前五名大模型中中国占四席。[^7]

中国模型的成功不是靠堆卡，而是靠三件事：

**第一，算法效率。** DeepSeek V3 采用的混合专家架构（MoE），让同等能力的推理成本下降了一个数量级——模型里有多个专家子模块，每次推理只激活其中一小部分，既保留能力又大幅降低成本。[^4]

**第二，开源策略。** DeepSeek、Qwen 的开源版本吸引了全球开发者，形成生态飞轮。

**第三，极致定价。** 中国厂商愿意以接近成本价换市场份额。主流国产模型 API 输入价格已进入 **0.5 元 / 百万 Token** 时代，仅为海外同性能模型的 1/10 到 1/20。[^8]

这意味着什么？**中国正在从「算力跟随者」变成「Token 定价破坏者」。**

但需要注意一个常见误读：在海外平台上提供服务的中国模型，推理计算通常部署在海外云节点（如新加坡、美国），而不是中国境内的数据中心。更准确的说法是：**中国的 AI 算法在出海，不是中国的电力在出海。**[^4]

对全球 Token 市场来说，中国模型的加入大幅加速了 Token 价格的下降。竞争越激烈，Token 越便宜，最终受益的是全球所有开发者和用户。

## 六、Token 到底该怎么定价？

Token 经济中已经出现了多种商业模式。[^4]

**按量计费**：用多少 Token 收多少钱。OpenAI、Anthropic 都走这个模式。简单透明，但用户会刻意减少调用。

**包月订阅**：ChatGPT Plus 每月 20 美元，不按 Token 计数。用户不再纠结每次调用的成本，使用量通常比按量计费高 5-10 倍。

**按价值收费**（还在萌芽）：不按消耗了多少 Token，而按创造了多少价值收费。比如 AI 帮你发现一个合同漏洞，避免了 1000 万损失，收你 10 万。利润率高到夸张，但前提是你能可靠衡量 AI 创造了多少价值。

**Token 期货**（正在萌芽）：企业预购未来的 Token 额度来锁定价格，像航空公司对冲油价一样。

史占中教授对此有更深层的分析：**Token 的定价正在发生一次范式跃迁。**[^7]

传统计算资源（CPU、GPU 按小时租用）的价格锚定的是硬件成本，遵循「成本加成」逻辑。但 Token 的定价已经脱离硬件成本，转向 **「价值定价」**——同一 GPU 集群运行不同版本的大模型，Token 价格可能相差数十倍，差异源自模型能力本身。

他进一步指出：**Token 兼具「消费品」（终端用户付费使用）和「生产资料」（开发者构建 AI 产品）的双重身份**，这在经济史上是极为罕见的。[^7]

当 Token 可以被期货交易的那一天，它就真正完成了从技术概念到经济基础设施的转变。

## 七、产业格局：谁在 Token 经济的牌桌上？

根据 IDC 的数据，2025 年中国公有云 MaaS 市场按调用量计算，**火山引擎占据了接近一半的份额**，其次是阿里云、百度智能云、硅基流动和移动云。[^6]

按营收计算，市场格局类似但略有不同：2025 年总营收约 **30.7 亿元人民币**，2026 年预计达 **186 亿元人民币**。[^6]

但 Token 经济不止是一家独大的故事。IDC 的报告揭示了一个关键转变：**竞争正在从「纯价格战」转向「价格 + 性能 + 工具链」的综合能力竞争。**[^6]

企业选型大模型时，Top 5 关注因素依次是：模型性能 → 安全合规 → 回答质量 → AI 平台可用性 → 成本效益。**成本反而排在第五位。**[^6]

这背后的逻辑是：企业先要确认「模型能不能用、安不安全、稳不稳定」，之后才会进入精细化的成本比较。但报告同时指出，随着 AI 应用渗透加深，算力短缺持续，**成本终将成为关键挑战**——只是现在还没爆发。

在经历前期的快速扩张之后，IDC 预期 Token 需求增长会逐渐回归理性，从陡峭的 J 型曲线过渡到更平缓的「次线性增长」阶段。2026—2028 年 Agent 规模化部署带来的脉冲式增长是关键拐点。[^6]

## 结语：Token 经济对普通人意味着什么？

七个问题，一条线索：

**Token 正在从一个技术术语，变成一种经济基础设施。**

它的消耗量两年增长了 1400 倍。它的生产成本每年下降一个数量级，但全球总支出反而翻倍——160 年前，杰文斯在煤炭上观察到的同一个悖论。它的需求正在从人类消费者扩展到 Agent。它的跨境流动绕过了一切物理运输。

我们今天能观察到的，不过是这场变革的开头几页。

对于 AI 从业者来说：Token 经济正在从底层逻辑上改变商业模式。理解 Token 的定价机制、供需结构、产业链分布，会成为 AI 领域的基本功。

对于普通用户来说：Token 的持续降价意味着 AI 工具会越来越便宜、越来越普及。三年内你现在觉得「AI 挺贵」的服务，可能会像今天的云存储一样几乎免费。

最后一个问题留给每个人自己：**你现在的工作中，有多少环节可以被拆解为「输入信息、处理、输出结果」？** 凡是符合这个模式的标准化流程，迟早会被 Token 化。留给人类的，是判断、创意、关系和不确定性——恰恰是单纯靠 Token 做不好的事。
